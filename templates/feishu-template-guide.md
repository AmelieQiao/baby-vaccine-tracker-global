# Feishu 多维表格模板搭建指南

## 表格结构（字段设计）

### 主表：疫苗接种记录表

| 字段名 | 字段类型 | 说明 |
|--------|---------|------|
| 疫苗名称 | 单行文本或单选 | **仅疫苗通用名**，不包含「第几剂」；剂次见「剂次」列 |
| 地区标注 | 单行文本 | 可选：🇨🇳🇺🇸🇩🇪 等区域标签（勿写进疫苗名称） |
| 孩子 / 宝宝 | 单选 | 填入孩子昵称 |
| 剂次 | 数字或文本 | 第几针（1/2/3/4）或 `年度` 等 |
| 人工备注 | 多行文本 | **仅家长手写**；自动化/AI **禁止改**此列 |
| 接种日期 | 日期 | 实际接种日期 |
| 品牌/厂家 | 单行文本 | 如"赛诺菲"、"辉瑞"、"国药" |
| 批号 | 单行文本 | 疫苗本上的批号（可选填） |
| 接种机构 | 单行文本 | 医院或卫生中心名称 |
| 状态 | 单选 | 已接种 / 待预约 / 过期未打 / 不适用 |
| 计划接种日期 | 日期 | 根据schedule计划的下次接种日期 |
| 备注 | 多行文本 | 不良反应、特殊情况等 |
| 是否免费 | 勾选 | 国家免疫规划=免费 |
| 参考标准 | 单选 | 国内标准 / 国际标准 / 两者 |

## 如何去除个人信息，做成模板

### 步骤（飞书多维表格内操作）

1. 打开你的多维表格
2. 点击右上角 **「···」** → **「复制多维表格」**
   - 新建一个副本，命名为 `baby-vaccine-tracker-TEMPLATE`
3. 在副本中：
   - 选中所有数据行（左上角空白处点击 → 全选）
   - 右键 → **「删除行」**（保留表头字段，清空所有记录）
   - 将孩子名称的单选选项改成 **「Child A」「Child B」**（通用化）
   - 删除任何备注字段里的个人诊所名称
4. 点击右上角 **「分享」** → 关闭个人分享链接，改为 **「任何知道链接的人可查看」**
5. 在 README 里放这个分享链接，让用户自己复制副本

### 或者：导出为 CSV 再整理

1. 在多维表格右上角点击 **「···」** → **「导出」** → **「导出为 CSV」**
2. 用 Excel/Numbers 打开，删掉所有数据行，保留第一行（字段名）
3. 另存为 `vaccine-record-template.csv`，放入 GitHub repo 的 `templates/` 文件夹

## 推荐视图配置

在飞书多维表格里设置以下视图（Views），方便使用：

### 视图1：按孩子分组（默认视图）
- 类型：表格视图
- 分组：按「孩子」字段分组
- 筛选：状态 ≠ 不适用
- 排序：接种日期 升序

### 视图2：待办日历
- 类型：日历视图
- 日期字段：计划接种日期
- 筛选：状态 = 待预约

### 视图3：本月到期
- 类型：表格视图
- 筛选：计划接种日期在「本月」内
- 筛选：状态 = 待预约 OR 过期未打

### 视图4：全部已完成
- 类型：表格视图
- 筛选：状态 = 已接种
- 排序：接种日期 升序

## 字段颜色建议（条件格式）

在表格视图 → 右上角「···」→「条件格式」：

- 状态 = **过期未打** → 行背景色：红色 🔴
- 状态 = **待预约** + 计划日期在30天内 → 行背景色：橙色 🟠
- 状态 = **已接种** → 行背景色：绿色 🟢
