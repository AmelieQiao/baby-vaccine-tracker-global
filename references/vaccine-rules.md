# Vaccine Rules: Intervals, Contraindications & Catch-Up Windows

Use this file when scheduling to check if proposed vaccine combinations and timing are valid.

---

## 1. Minimum Intervals Between Doses (同种疫苗的最短间隔)

| Vaccine | Minimum interval between doses |
|---------|-------------------------------|
| HepB | 4 weeks (D1→D2); 8 weeks (D2→D3); 16 weeks total (D1→D3) |
| DTaP | 4 weeks (D1→D2, D2→D3, D3→D4); 6 months (D4→D5 booster) |
| IPV | 4 weeks (D1→D2, D2→D3) |
| PCV13 | 4 weeks (D1→D2, D2→D3); 8 weeks (D3→D4 booster, must be ≥12m) |
| Hib | 4 weeks (D1→D2, D2→D3); 8 weeks (D3→D4 booster, must be ≥12m) |
| MMR | 4 weeks if two doses needed close together (minimum); 4 weeks between live vaccines |
| Varicella | 3 months (D1→D2 per China schedule); 4 weeks minimum absolute |
| Rotavirus (Rotarix) | 4 weeks (D1→D2); series complete by 24 weeks of age |
| Rotavirus (RotaTeq) | 4 weeks (between each); series complete by 32 weeks of age |
| HepA (inactivated) | 6 months (D1→D2); minimum 5 months |
| JE (live) | 1 year (D1→D2 booster) |
| MenA | 3 months (D1→D2); both before 18m |
| Flu | 4 weeks if two doses in first year; annual thereafter |
| EV71 | 1 month (D1→D2) |
| HPV (2-dose) | 6 months (D1→D2, for those starting before 15yr) |
| HPV (3-dose) | 2 months (D1→D2), 4 months (D2→D3), 6 months total (D1→D3) |

## 2. Live Vaccine Rules (减毒活疫苗间隔规则)

**CRITICAL RULE**: Any two injectable live vaccines must be given EITHER:
- On the SAME day (同一天), OR
- At least **28 days apart** (≥4 weeks)

**Live vaccines in this skill:**
- MMR (麻腮风)
- Varicella (水痘)
- BCG
- JE live (乙脑减毒活疫苗)
- Rotavirus oral live (轮状 oral, Rotarix, RotaTeq, 兰州)
- Yellow Fever (travel)

**Note**: Oral rotavirus is live but oral route — some guidelines treat it differently, but to be safe, observe 28-day rule from injectable live vaccines.

**Inactivated vaccines**: No minimum interval with each other or with live vaccines (they can be given same day at different sites).

## 3. Same-Day Co-Administration (同一天可以一起打的组合)

**General rule**: Multiple inactivated vaccines can be given on the same day (different limbs/sites).

| Combination | OK same day? | Notes |
|-------------|-------------|-------|
| PCV13 + DTaP + IPV | ✅ Yes | Common combination at 2m, 4m, 6m |
| PCV13 + Hib | ✅ Yes | |
| Flu + any inactivated | ✅ Yes | |
| MMR + Varicella | ✅ Yes | Both live, same day is fine |
| MMR + BCG | ✅ Yes (same day) | Cannot give 1 day apart — must be same day or 28d+ |
| MMR + Flu (inactivated) | ✅ Yes | |
| MMR + Rotavirus (oral) | ✅ Yes (same day) | Use caution: some clinics prefer to separate |
| HepB + anything inactivated | ✅ Yes | |
| EV71 + inactivated vaccines | ✅ Yes | |
| DTaP + HepB + IPV | ✅ Yes (effectively what 六联 does) | |

**Maximum vaccines per visit**: No absolute limit, but most clinics in China limit to 2–3 injections per visit to manage pain and observation time. Factor this into scheduling.

**Practical clinic limit**: 2 injections typically, occasionally 3 (one per limb + one more). Discuss with the specific clinic.

## 4. Contraindications & Precautions (禁忌和注意事项)

### Absolute Contraindications (不能打)

| Situation | Vaccines to avoid |
|-----------|------------------|
| Severe allergy to vaccine component (anaphylaxis) | That specific vaccine — consult allergist |
| Encephalopathy within 7 days of prior DTaP dose | Further DTaP doses |
| Immunocompromised (先天性免疫缺陷, on chemo, etc.) | All live vaccines (MMR, Varicella, BCG, JE live, Rotavirus) |
| HIV positive | BCG, Yellow Fever; check others with doctor |
| Pregnancy | Live vaccines; though infants are not pregnant — flag for mothers |
| Allergy to eggs (severe) | Some flu vaccines — check brand. Recombinant flu vaccines are egg-free (e.g., Flublok) |
| Allergy to gelatin | MMR, Varicella (gelatin stabilizer) — consult doctor |

### Temporary Precautions (暂时推迟)

| Situation | Action |
|-----------|--------|
| Fever (发烧) > 38.5°C | Postpone until recovered, typically 1 week after fever resolves |
| Acute illness (急性病) | Postpone until recovered |
| Within 3 months of blood transfusion or IVIG | Postpone live vaccines (MMR, Varicella) |
| Recent illness (cold/diarrhea) | Wait 1–2 weeks after full recovery |
| Eczema flare (湿疹急性期) | Postpone — confirm with doctor |
| Just had surgery | Postpone — confirm with doctor |

## 5. Catch-Up Windows (补种窗口期)

These are the LATEST ages at which a vaccine series can still be started or completed meaningfully.

| Vaccine | Last age to START | Notes |
|---------|-------------------|-------|
| Rotavirus (Rotarix) | 15 weeks for Dose 1 | Series must complete by 24 weeks |
| Rotavirus (RotaTeq) | 12 weeks for Dose 1 | Series must complete by 32 weeks |
| 国产轮状 (兰州) | 6 months for Dose 1 | Can continue until 3 years |
| EV71 (hand-foot-mouth) | 5 years (before 5th birthday) | No benefit after age 5 |
| BCG | Typically birth; catch-up possible but requires TB test first | |
| Hib | If starting at 12–15m, only 1 dose needed | |
| PCV13 | If starting at 12–23m, 2 doses needed; 24m+: 1 dose | |
| MMR | No upper age limit for catch-up | |
| HepB | No upper age limit for catch-up | |
| DTaP | Switch to Tdap at 7yr+ | |
| Varicella | No upper limit if no history of disease | |
| HPV | Ideally before sexual debut; up to 45yr (China approval) | |
| Flu | Annual — no catch-up concept, just get each year |

## 6. Brand Consistency Rules (品牌延续性)

**RULE**: For vaccines with multiple doses, try to complete the series with the SAME brand/manufacturer.

| Vaccine | Brand consistency important? | Notes |
|---------|------------------------------|-------|
| DTaP (as part of combo) | ✅ Yes — strongly preferred | 五联, 六联, and free DTaP are NOT interchangeable within a series ideally |
| Rotavirus | ✅ YES — CRITICAL | Rotarix (2-dose) and RotaTeq (3-dose) are NOT interchangeable. Must complete series with same brand. |
| PCV | Preferred same brand | Prevnar 13 and Synflorix (10-valent) not interchangeable; Prevnar 13 and 沃森PCV13 are considered interchangeable in practice |
| Hib | Preferred same brand | Act-Hib vs combo vaccines — coordinate with clinic |
| HepB | Flexible | Can switch brands between doses |
| MMR | Flexible | All brands equivalent |
| Flu | Flexible | Annual, new vaccine each year anyway |
| HPV | ✅ YES — CRITICAL | Must complete with same brand (Gardasil-9 vs Cervarix not interchangeable) |

**When brand mismatch occurred**: Note it in the record and consult clinic. In most cases, the series can still proceed — immunogenicity is usually maintained. Document in vaccine booklet.

## 7. Post-Vaccination Observation (接种后观察)

- **Mandatory observation**: 30 minutes at clinic after every vaccine (Chinese regulation)
- Bring a snack, water, comfort toy for the child
- **Watch for**: Anaphylaxis (rare, within 30 min), local redness/swelling (normal), low-grade fever (normal within 24–48h)
- **MMR-specific**: Mild rash possible 5–12 days after; low fever normal
- **BCG-specific**: Local ulceration at injection site is normal and expected (4–8 weeks after)
- If fever > 39°C or lasts > 3 days post-vaccine: consult doctor

## 8. Scheduling Algorithm (for the AI to follow)

When generating a batched schedule:

1. **List all pending vaccines** per child with urgency level (overdue / due this month / due next 3m / future)
2. **Group by date window**: Can these vaccines be given on the same day?
   - Check co-administration rules above
   - Max 2–3 injections per visit (1 oral counts differently)
3. **Resolve live vaccine conflicts**: If two live vaccines needed, either same day or 28d apart
4. **Check intervals**: From last dose of same vaccine, ensure minimum interval met
5. **Match children's schedules**: Try to batch both children on same clinic day
6. **Output**: Month-by-month schedule, trip count, per-child per-trip vaccine list
7. **Flag**: Any vaccines that CANNOT wait (overdue catch-up) vs. those with flexibility
